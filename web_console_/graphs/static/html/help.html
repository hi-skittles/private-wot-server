Graphs Help

The Graphs module is a set of tools for the graphical display of live and historical server statistics, collected by the StatLogger process.

The StatLogger process is responsible for logging user-configurable collections of server metrics over time to either (or both) relational and Carbon-based metric databases. For further information, see the "StatLogger" chapter of the Server Operations Guide.

This version of the Graphs module is intended as a developer preview to demonstrate the use of Graphite/Carbon as an eventual replacement for the existing StatLogger relational database and existing Graphs module.

Accordingly, some features of the original Graphs module are not included in this version.

Viewing Graphs

BigWorld server metrics can be visualised from the perspective of process type (via the Processes page) or machine (via the Machines page). These views present an overview of all machines and processes. You can narrow these views down to specific machines, processes, or machine/process groups by following the links in the chart titles.

Toolbar

You can use the toolbar above the graph display to change the view.

Graph Display

One or more graphs are displayed in this area. All graphs displayed are synchronised to the same time and display the same time range.

If the number of subjects for graphing exceeds the maximum number of graphs, graph series are consolidated into groups alphabetically. When grouped, graph titles take the form "<first-member-in-group> - <last-member-in-group>". Specific graphs within a group may be "drilled into" by clicking the graph title.

You can drill back up to previous graph view states by clicking the breadcrumb links to the left of the toolbar.

Setting the Graphing Period

To specify the graphing period, do any of the following:

  • Click the button and specify an end date and time for the graphing period. Use the drop-down menu beside this button to select the period of time to graph.
  • Hold down Shift and drag across the graph to select a specific time period.
  • Pan left (backwards in time) or right (forwards in time) by dragging any of the displayed graphs left or right.
  • Hold down Shift and use the mouse scroll wheel to zoom in (increasing the graphing period) or out (decreasing the graphing period).

Live Mode

You can snap the current graphing period to the current time by clicking the Live mode button.

Live mode will poll for new data no more often than the current point resolution. For example, data viewed at two second resolution will be polled once every two seconds.

Note: Any manual panning action while in live mode turns live mode off.

The Legend

Display of the legend pane is off by default, and can be toggled via the Legend button. When shown, the legend pane displays the names and values of the graph currently under the mouse pointer.

Note: The colours of metrics are as defined in the StatLogger preferences file ("preferences.xml"). For more information on this file, see "Process Statistic Configuration" in the Server Operations Guide.
